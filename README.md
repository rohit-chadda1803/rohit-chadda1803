### 🔥 Featured Projects

**🌟 NodeBase — Visual Workflow Automation Platform**  
A powerful **node-based workflow automation tool** where users can drag-and-drop nodes on an interactive canvas, connect them, set triggers, and execute complex automated flows.  

Built with modern technologies including **TypeScript, tRPC, Inngest** for background jobs, **Sentry** for monitoring, and **Polar** for seamless integrations. It demonstrates my interest in building intuitive developer tools and complex state management on the frontend.

**🐍 CobraKai — Intelligent Image Puzzle Solver**  
A computer vision application that extracts text and patterns from puzzle images using **OpenCV + Tesseract OCR**, processes the data, and solves the puzzle automatically.  

Combines image processing, OCR accuracy tuning, and algorithmic solving — showcasing my ability to integrate classical computer vision with problem-solving logic.

**🏨 Abook — Smart Room Booking System**  
Full-featured accommodation booking platform with search, availability checking, reservation management, and clean user interface.

**📚 StudyBud**  
Collaborative study platform built with Django. Enables students to create study groups, share resources, and stay organized together.

**🛒 MyCart — Full Django E-Commerce Platform**  
Complete online store featuring product browsing, cart system, blog for product reviews, order tracking, and admin dashboard. Demonstrates end-to-end e-commerce architecture and real-world Django best practices.

**🎵 Spotify Clone**  
Modern UI/UX replication of Spotify with focus on responsive design and core music player functionalities.
